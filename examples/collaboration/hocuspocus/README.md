# SuperDoc + Hocuspocus

Self-hosted collaboration using [Hocuspocus](https://hocuspocus.dev/).

**Docs:** [Hocuspocus Guide](https://docs.superdoc.dev/guides/hocuspocus)

## Getting started

```bash
npm install
npm run dev
```

This starts both the Hocuspocus server (ws://localhost:1234) and the React client (http://localhost:3000). Open two browser tabs to see real-time collaboration.
